# ONIQ Platform

Marketplace de logiciels métier pour professionnels indépendants et TPE.

## Déploiement rapide sur Vercel

1. Installe Node.js (https://nodejs.org)
2. Dans ce dossier, lance :
   ```
   npm install
   npm run build
   ```
3. Va sur https://vercel.com → "Add New Project"
4. Glisse le dossier `dist/` généré, ou connecte ton GitHub
5. Vercel déploie automatiquement ✓

## Développement local

```
npm install
npm run dev
```

Ouvre http://localhost:5173
